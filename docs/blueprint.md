# **App Name**: C-Pop Now

## Core Features:

- Homepage Carousel: Display trending C-Dramas via a horizontally-scrolling carousel of poster images.
- Drama Grid View: Display dramas in a grid format, adapting to mobile, tablet and desktop screen sizes.
- Genre Filtering: Filter dramas by genre using pill-shaped filter buttons.
- Drama Details Page: Dedicated page for each drama containing full-sized poster image, expandable synopsis and episode lists

## Style Guidelines:

- Primary color: Chinese Red (#9B1C1C) to convey cultural significance and boldness.
- Secondary color: Gold (#F2C94C) as an accent for interactive elements and highlights.
- Neutral colors: White (#FFFFFF) for backgrounds and Black (#000000) for text, borders, and shadows, ensuring high contrast and legibility.
- Headings: 'Space Grotesk' (sans-serif) in uppercase and bold for a modern, high-impact presentation.
- Body text: 'Inter' (sans-serif) for clear and readable descriptions and details.
- Pop art elements: star icons and bold labels to highlight ratings and trending dramas.
